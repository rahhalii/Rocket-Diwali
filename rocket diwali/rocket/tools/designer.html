Skyrocket.js - Rocket explosion shape designer

Use this tool to design firework explosions. Place particles by left-clicking with the mouse around the center point (which is where the skyrocket will fly to). You can select existing particles and alter their properties as well as dragging them to a new location.

Canvas not supported, please try a different web browser.
